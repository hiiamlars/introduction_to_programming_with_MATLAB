# Introduction to Programming with MATLAB (Assignments)

This repository contains the assignements developed during the course *"Einführung in die Programmierung mit MATLAB"* (Introduction to Programming with MATLAB), taught by Prof. Dr. Felix Wichmann as a guest lecturer at the University of Bern in the fall semester 2025. 

## Covered Topics

- **Introduction to MATLAB**: This section covered the basics of the MATLAB environment, vectorization, and data structures like matrices and cell arrays.
- **Functions**: This topic focused on creating reusable code architectures, managing input and output arguments dynamically, and documenting this architecture.
- **Defensive coding**: This module emphasized writing crash-resistant code like implementing strict input validation, assertions and informative messages.
